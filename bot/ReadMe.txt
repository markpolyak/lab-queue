Для того, чтобы запустить проект необходимо:
1) gspread==5.4.0 (установка: pip3 install gspread)
2) pyTelegramBotAPI==4.6.0 (установка: pip install pyTelegramBotAPI)
3) база данных sqlite 

Используемые файлы:
1. файл config.ру содержит токен и информация о Google-таблице
2. database.db - база данных, в которой содержиться группа, фамилия и имя зарегистрированных студентов
3. в файле database.ру содержаться функции для работы с базой (чтение и сохранение данных)
4. googletable.ру - содержит функции для работы с Google-таблицей (открытие таблицы, чтение данных, запись студентов и приоритетности в очередь, удаление и добавление очереди)
5. main.ру - главный загрузочный файл, который связывает все файлы воедино
